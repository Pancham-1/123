Portfolio

Portfolio website is built using HTML, CSS, and JavaScript to create a visually appealing and responsive design. The HTML structures the content, while CSS ensures a clean and modern aesthetic with a focus on readability and user experience. JavaScript adds interactivity, enhancing navigation and dynamic content display. The website features a minimalist layout, smooth transitions, and an intuitive user interface. It is fully responsive, ensuring compatibility across various devices and screen sizes.
